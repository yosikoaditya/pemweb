Berikut ini Praktikum CSS

semua praktikum di pdf saya buat 1 file saja. tetapi akan mencangkum seluruhnya. bisa dicek di kodenya langsung. saya tandai

Praktikum 1

internal css, adalah kode css yang ditulis dengan tag style. contohnya bisa dicek di kodenya

Praktikum 2

eksternal css, adalah kode css diluat kode induk. misalnya pada fong tag p yang saya beri 20px

Praktikum 3

eksternal css, adalah kode ccs yang berada diluar file induk/ utama, dan akan dipanggil nantinya. Contohnya bisa dicek di kode. disini saya mencontohkan tag p dibuat 20px

Praktikum 4

Inline Css, adalah kode yang langsung ditulis pada atribut elemen html. contohnya pada background kuning yang saya taruh di body

Praktikum 5

Selektor tag css, adalah css yang jika dibentuk maka akan memengaruhi semua selektor yang sama. contohnya selektor h1 saya kasih css di style

Praktikum 6

Selektor class css, adalah kode css yang terdapat pada tag style yang diberi tanda . didepanya. contohnya, box warna merah ini lho pakai selector css

Praktikum 7

Selektor id, ini hampir sama seperti class, tapi yang ini boleh digunakan sekali saja dan ada tanda #

Praktikum 8

selektor atribut. misalnya dibawah ini, akan mengubah elemen imput yang atributnya type=text

Praktikum 9

selektor universal, adalah css yang kodenya berlaku untuk semuanya. contohnya semua font disini menggunakan ariial
